# 📱 VidSnap — Social Media Video Downloader

A stunning mobile-first web tool to download HD videos (with audio) from YouTube, Instagram, TikTok, Twitter/X, Facebook, Reddit, Vimeo and 100+ other sites.

---

## 🚀 Quick Start

### 1. Install yt-dlp (Required)
```bash
pip install yt-dlp
```

### 2. Start the server
```bash
node server.js
```

### 3. Open in browser
- **Desktop:** http://localhost:3001
- **Mobile (same WiFi):** http://YOUR-PC-IP:3001

---

## 📋 Supported Sites
YouTube, Instagram, TikTok, Twitter/X, Facebook, Reddit, Vimeo, Dailymotion, Pinterest, Snapchat, and 100+ more (powered by yt-dlp).

## ✨ Features
- **HD Quality** — up to 4K if available
- **With Audio** — video and audio merged into one MP4
- **No Watermark** — clean downloads
- **Quality Selector** — pick 1080p, 720p, 480p, etc.
- **Mobile Friendly** — works great on phone browsers

## 🔧 Dependencies
- Node.js 18+
- Python + yt-dlp (`pip install yt-dlp`)
- ffmpeg (for merging video+audio — install from https://ffmpeg.org)

## ⚠️ Note
For best results (merging HD video + audio), install ffmpeg and add it to your PATH.
